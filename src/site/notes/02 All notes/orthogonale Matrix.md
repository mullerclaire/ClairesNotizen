# orthogonale Matrix

Eine Matrix $S \in \mathrm{M}_{n}(\mathbf{R})$ heißt **orthogonal**, falls $S \cdot{ }^{t} S=E$ gilt; 

äquivalent dazu ist die Bedingung $S^{-1}={ }^{t} S$.
