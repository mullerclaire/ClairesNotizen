# hermitesche Matrix
Eine Matrix $A \in \mathrm{M}_{n}$ (C) heißt hermitesch, falls $A={ }^{t} \bar{A}$ gilt. Wir setzen
$$
\operatorname{Herm}_{n}(\mathbf{C})=\left\{A \in \mathrm{M}_{n}(\mathbf{C}) \mid A \text { hermitesch }\right\} .
$$
