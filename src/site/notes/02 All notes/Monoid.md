# Monoid
$(M, *, e)$ mit $M$ eine Menge, $*$ eine innere zweistellige Verknüpfung $$*:M\times M\to M, \quad (a,b)\mapsto a*b, $$ $e$ ein Element aus $M$ mit den Eigenschaften: 
1. Assoziativität von $*$,
2. $e$ ist neutrales Element.
