# Ring von Figuren 
Seien $\mathcal{E}=\{Q\subset\R^n|Q \text{ halboffener Quader}\}$
$\mathcal{R}=\{A\subset\R^n| A \text{ Figur}\}$. 

Dann ist $\mathcal{R}$ ein Ring, aber keine $\sigma$-Algebra.
