# $\sigma$-Algebra
Ein nichtleeres Mengensystem $\mathcal{S}\subset 2^X$ heißt ==$\sigma$-Algebra==
falls für $A_1, A_2,... \in\cal S$ gilt

- $\displaystyle\bigcup_{i=1}^{\infty} A_i\in\cal S$
- $A_i^c\in\cal S$.
