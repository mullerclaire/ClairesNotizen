# $\mu$-Nullmenge
In einem [[02 IMP Notizen/Maßraum|Maßraum]] $(X,\mathcal A, \mu)$: 
die $N\in\mathcal A$ mit $\mu(N)=0$ heißen $\mu$-==Nullmengen==. 

___
# $\mu$-Nullmenge
```ad-remember

$\mathcal{N}_\mu=\{N\in 2^X\mid\exists C\in\mathcal{A}, N\subset C, \mu(C)=0\}$

```
