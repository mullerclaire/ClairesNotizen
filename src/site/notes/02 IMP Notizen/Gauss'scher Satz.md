# Gauss'scher Satz
```ad-equation
title: Gauss'scher Satz
Sei $V\in \R$ umschlossen von der Oberfläche $S(V)$, $\vec{a}(\vec{x})$ differenzierbares Vektorfeld. 

Dann gilt: 
$$\int_V\mathrm{d}V\,\text{div}\,\vec a(\vec x)=\oint_{S(V)}\mathrm{d}\vec f\cdot\vec a(\vec x)$$

```

```ad-remark
title: Intuition zum Gaußschen Satz
Der Fluss von $\vec a$ bestimmt sich aus $\vec\nabla\cdot\vec a$ im Volumen. ![[Capture d’écran 2022-02-03 à 16.46.26.png]]

```
